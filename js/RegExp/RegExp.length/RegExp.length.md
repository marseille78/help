# RegExp.length

**RegExp.length** - повертає кількість параметрів.

## Синтаксис:

```javascript
RegExp.length
```

## Опис:

**length** властивість об'єкту [`RegExp`](../RegExp.md) яка містить кількість параметрів скільки приймає конструктор.

## Приклад:

```javascript
alert( RegExp.length ); //2
```
